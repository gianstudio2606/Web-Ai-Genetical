# 🧬 Genetical AI

AI Chat dengan model **Llama 3.3 70B Abliterated** (tanpa sensor). Bisa menganalisis gambar/video dan membuat kode.

## ✨ Fitur

- 💬 Chat AI dengan model Llama 3.3 Abliterated
- 📸 Upload gambar/video untuk dianalisis
- 💻 Mode coding - minta AI membuat kode
- 🎨 Desain modern dengan Google Material Icons
- 📱 Responsif untuk HP dan desktop

## 🚀 Cara Pakai

1. Buka URL: `https://nexus-ai.vercel.app`
2. Tanyakan apa saja, atau upload file
3. Aktifkan mode coding dengan tombol `</>` untuk minta kode

## 🛠️ Teknologi

- HTML + CSS + JavaScript (Vanilla)
- Google Material Icons
- Font Inter
- Featherless AI API (Llama 3.3 70B Abliterated)

## ⚠️ Peringatan

Model ini memiliki filter keamanan yang sangat berkurang. Gunakan dengan bijak dan bertanggung jawab.
